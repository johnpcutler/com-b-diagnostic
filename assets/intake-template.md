# Diagnostic Intake

Use this template to structure the input for a COM-B behavioral diagnosis.

---

## Target behavior

_What should people be doing? Be specific about the action, frequency, and context._

## Current reality

_What are people actually doing instead? Include observable patterns, not just absence of the target behavior._

## Actors

_Who is involved? List roles and their relationship to the target behavior._

| Actor / Role | Relationship to target behavior |
|-------------|-------------------------------|
| | |

## Context

_What is the environment? Include relevant organizational, technical, and social factors._

- **Recent changes:** _(reorgs, new tools, policy shifts, leadership changes)_
- **Existing supports:** _(training, tooling, documentation, incentives already in place)_
- **Known constraints:** _(time pressure, competing priorities, resource limits)_

## Scope

- [ ] Single behavior, single actor group
- [ ] Single behavior, multiple actor groups (multi-actor diagnosis)
- [ ] Multiple related behaviors

## Prior attempts

_What has already been tried? What happened?_

## Success criteria

_What would "better" look like? How would you observe it?_
