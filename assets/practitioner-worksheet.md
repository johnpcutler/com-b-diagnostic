# Practitioner Worksheet

Internal working document for synthesis and intervention design.
Arrive here after completing a thorough lens analysis (Step 2)
using the lens files directly.

## Prerequisites

- Behavior definition complete (see [behavior-canvas.md](behavior-canvas.md))
- Lens analysis complete: you have read the relevant sub-lenses in
  [capability-lenses.md](../references/lenses/capability-lenses.md),
  [motivation-lenses.md](../references/lenses/motivation-lenses.md),
  [physical-opportunity-lenses.md](../references/lenses/physical-opportunity-lenses.md),
  [social-opportunity-lenses.md](../references/lenses/social-opportunity-lenses.md)
  and identified which dimensions are active, their positions, and evidence.
- Consult [lens-map.md](../references/lenses/lens-map.md) for navigation.

## 2. Per-lens synthesis

For each COM-B branch where you found active dimensions, complete the
section below. The goal is not to list every dimension -- it is to
build a narrative that connects dimensional positions to intervention
reasoning.

**Instructions:**
- **Active sub-lenses:** List only the sub-lenses where you found
  meaningful signal -- either signal of what to try (gaps that
  interventions can close) or signal of what not to try (conditions
  that make certain approaches ineffective or counterproductive).
  Use shortcodes from lens-map.md.
- **Key dimensional findings:** For each active sub-lens, name the
  dimensions that matter most, their positions (low/mid/high), and
  the evidence that supports each position. Focus on dimensions that
  are gaps or that interact with findings in other lenses.
- **Pattern guidance conclusions:** Read the Pattern guidance block
  for each active sub-lens. Based on the dimensional positions you
  found, state which shaping notes apply, which redirecting notes
  are triggered, and what that means for intervention in this
  situation. This is where the diagnosis becomes actionable -- do
  not skip it.
- **Relevant BCTs:** For each active sub-lens, list the specific
  BCTs that the Pattern guidance block indicates given your
  dimensional positions. Use the Refs index line to quickly identify
  candidates, then select the ones that fit this situation. Cite as
  BCT.n.m with the technique name.

## 3. Cross-lens interactions

Identify how findings from different lenses interact:
- **Tensions:** where lenses conflict (e.g. capability present
  but social environment punishes it)
- **Reinforcements:** where lenses point to the same bottleneck
- **Prerequisites:** where one lens must be addressed before
  another can work (drawn from Pattern guidance redirecting notes)
- **Highest leverage:** which 2-3 findings would unlock the most
  change if addressed
