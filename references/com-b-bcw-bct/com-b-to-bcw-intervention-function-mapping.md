# OCR: COM-B to BCW Intervention Mapping

## Description
Three tables bridge COM-B diagnosis to specific behaviour change techniques. The first maps each COM-B barrier to its relevant BCW intervention functions (per Michie, Atkins & West, 2014). The second defines the nine intervention function abbreviations. The third links each intervention function to the BCT groupings that typically implement it, with deep links into the full taxonomy.

## Extracted Content

### COM-B Problem -> Relevant Intervention Functions (BCW)

| COM-B Problem | Relevant Intervention Functions (BCW) |

| Physical Capability (PHC) | Training, Enablement |
| Physical Opportunity (PO) | Restriction, Environmental Restructuring, Enablement |
| Social Opportunity (SO) | Restriction, Environmental Restructuring, Modelling, Enablement |
| Reflective Motivation (RM) | Education, Persuasion, Incentivisation, Coercion |
| Automatic Motivation (AM) | Persuasion, Incentivisation, Coercion, Training, Environmental Restructuring, Modelling, Enablement |

### BCW Intervention Function Abbreviations

| Abbrev | Full Name | What It Means |
|---|---|---|
| ED | Education | Increase knowledge or understanding. |
| TR | Training | Build skills or procedural know-how. |
| PE | Persuasion | Shift beliefs or emotions; motivate through argument, framing, salience. |
| INC | Incentivisation | Reinforce via rewards, recognition, positive consequences. |
| COE | Coercion | Apply pressure, consequences, or negative expectancies (rare for tools). |
| RE | Restriction | Limit options to reduce competing behaviours (tools rarely use this). |
| ER | Environmental Restructuring | Change physical or social environment; reduce friction; introduce cues. |
| MO | Modelling | Show examples of good behaviour or patterns to imitate. |
| EN | Enablement | Remove barriers beyond education/training; make behavior possible/easier. |
